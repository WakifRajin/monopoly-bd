Update 0.0.6 - Winner Screen Upgrade

Cleaner endgame flow

  • Winner screen now always stays above overlays, including bank auctions.
  • Added winner actions: View Board and Play Again.
  • Added a compact leaderboard showing other players and their final cash.
  • Post-game board view is now read-only to prevent accidental extra actions.

Update 0.0.5 - Bankruptcy and Online Reliability

Debt resolution stability

  • Improved AI debt handling so AI can sell, mortgage, or declare bankruptcy without getting stuck.
  • Debt prompt state now syncs into game snapshots for better online consistency.
  • Center turn messaging now shows active debt resolution status in online matches.
  • Added winner fallback checks so endgame winner display appears more reliably online.

Update 0.0.4 - Home and Patch Notes

New discoverability for updates

  • Added a small circular What's New button at the bottom of the home screen.
  • Patch notes are now tracked in update-version format instead of dates.

Update 0.0.3 - Auction System Revamp

Gameplay and menu improvements

  • Added a new Auction System toggle in lobby settings (On or Off).
  • Auction opening bid no longer starts at zero.
  • Opening bid now starts around 60% to 80% of the property base price.
  • Auction base price is now shown inside the auction panel.
  • When auctions are disabled, unsold properties remain with the bank instead of opening auctions.

Update 0.0.2 - Online Sync and Stability

Host room settings and rule consistency

  • Auction settings now sync correctly in online rooms.
  • Host controls are respected for lobby game-setting updates.
  • Bankruptcy and leave-liquidation flows now follow auction on/off state consistently.
  • Auction states are validated and normalized during online snapshot hydration.

Update 0.0.1 - UI Updates

Auction panel improvements

  • Auction panel has improved display content for bidding context.
  • Current bid, leader, bidder cash, and base price are shown together for clarity.

This page can be updated with each release to track balancing and bug fixes.
