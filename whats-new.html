Latest Update

Update 0.0.8 - UI Updates and Custom Board Editor

Homepage polish, board tools, audio, and reliability

  • Redesigned the home screen with the new layout and structured Play, Explore, and Dev sections.
  • Updated home action cards and typography for better readability on desktop and mobile.
  • Added and exposed the Custom Board Editor flow from home for easier board experimentation and setup.
  • Expanded in-game audio with event-based sound effects and background music support.
  • Added sound controls in settings so players can toggle SFX and music and adjust volume.
  • Applied general bug fixes across gameplay flow, including turn-state, movement-card, and UI lock edge cases.
  • Improved online stability with better snapshot handling, AI takeover continuity, and sync reliability.
  • Refined the Report button so it stays compact and fully inside the viewport.
  • Redesigned this Credits and Patch Logs page to match the new homepage visual language.
Previous Updates

Update 0.0.7 - Classic Board and Stability Fixes

Rule parity and turn-flow reliability

  • Added the new Classic board with original Monopoly-style economy: $200 GO, $1500 start money, classic property values, classic railroads and utilities, and $50 jail bail.
  • Added theme-specific Chance and Community card profiles so Classic uses original-scale card values.
  • Utility rent logic is now theme-aware and ownership-safe, fixing incorrect utility rent edge cases.
  • Fixed async turn-flow race issues that could cause mixed turn prompts or wrong-player action locks.
  • Fixed resolving-card and movement-lock stuck states by forcing immediate UI unlock after effect or move completion.
  • Improved Classic AI economy behavior: fewer unnecessary mortgages, better unmortgage recovery, and no auction attempts when auctions are disabled.

Update 0.0.6 - Winner Screen Upgrade

Cleaner endgame flow

  • Winner screen now always stays above overlays, including bank auctions.
  • Added winner actions: View Board and Play Again.
  • Added a compact leaderboard showing other players and their final cash.
  • Post-game board view is now read-only to prevent accidental extra actions.

Update 0.0.5 - Bankruptcy and Online Reliability

Debt resolution stability

  • Improved AI debt handling so AI can sell, mortgage, or declare bankruptcy without getting stuck.
  • Debt prompt state now syncs into game snapshots for better online consistency.
  • Center turn messaging now shows active debt resolution status in online matches.
  • Added winner fallback checks so endgame winner display appears more reliably online.

Update 0.0.4 - Home and Patch Notes

New discoverability for updates

  • Added a small circular What's New button at the bottom of the home screen.
  • Patch notes are now tracked in update-version format instead of dates.

Update 0.0.3 - Auction System Revamp

Gameplay and menu improvements

  • Added a new Auction System toggle in lobby settings (On or Off).
  • Auction opening bid no longer starts at zero.
  • Opening bid now starts around 60% to 80% of the property base price.
  • Auction base price is now shown inside the auction panel.
  • When auctions are disabled, unsold properties remain with the bank instead of opening auctions.

Update 0.0.2 - Online Sync and Stability

Host room settings and rule consistency

  • Auction settings now sync correctly in online rooms.
  • Host controls are respected for lobby game-setting updates.
  • Bankruptcy and leave-liquidation flows now follow auction on and off state consistently.
  • Auction states are validated and normalized during online snapshot hydration.

Update 0.0.1 - UI Updates

Auction panel improvements

  • Auction panel has improved display content for bidding context.
  • Current bid, leader, bidder cash, and base price are shown together for clarity.
